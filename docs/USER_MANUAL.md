# User Manual

Welcome to CodeVault Team — your team's private, realtime code storage vault.

## Signing in
Go to the app URL → **Sign In** with the email/password your admin created for you. Use **Forgot password?** if needed. Check "Remember me" to stay signed in on this device.

## Dashboard
Your home screen — shows root folder/file counts, total languages, storage used, your favorites, and a quick "New Project" shortcut.

## Languages
A tag-like list of every technology your team uses (React, Django, PHP...). Click **Add Language** to create a new one with a color. Click a language card to see everything tagged with it (via Tags).

## Projects
High-level project entries (name, description, linked language) — a lightweight way to track what you're building, separate from the folder/file structure itself.

## Folders & Files (`Files` in the sidebar)
This is the core of the vault:
- **New Folder** — create a folder at the current level.
- **Upload** — pick one or more files.
- **Upload Folder** — pick an entire folder from your computer; the folder structure is recreated automatically inside the vault.
- **Drag & drop** — drag files or a `.zip` anywhere onto the page. ZIPs are automatically extracted into matching nested folders.
- Click a folder to navigate into it — breadcrumbs at the top let you jump back up.
- Click a file to **preview** it (images, PDFs, and code/text render inline; other types offer a download).
- Every card has a **⋯ menu**: Rename, Tags, Add to Collection, Comments (folders) / Preview, Edit, Download (files), Favorite, Delete.

## Code Editor
Opens automatically when you click **Edit** on a text/code file. Features: syntax highlighting, autosave every 2 seconds (plus a manual **Save** button), version history (side panel — click any past version to load it back into the editor, then Save to commit), fullscreen mode, light/dark theme toggle, and copy-to-clipboard.

## Snippets
For small reusable bits of code that don't need a full file — paste code, tag it with a language, and it's instantly available to the whole team with one-click copy.

## Search
The search bar in the top bar (or the Search page) looks across every folder and file name instantly.

## Tags & Collections
- **Tags** — colored labels you can attach to any folder or file (e.g. "ERP", "College", "Urgent") from its ⋯ menu. Create new tags on the fly.
- **Collections** — named groups of folders that might span multiple languages/projects (e.g. "Client Work", "Learning"). Add a folder to a collection from its ⋯ menu.

## Favorites
Star any folder or file to pin it here for one-click access later.

## Comments
Leave comments on a file (inside its preview) or a folder (via ⋯ → Comments) — visible to the whole team in realtime.

## Notifications
The bell icon shows a live badge when a teammate uploads, creates, or deletes something. Click through to see the full list; click a notification to mark it read.

## Recycle Bin
Nothing is deleted immediately — soft-deleted folders/files land here first and can be restored anytime, until your admin's configured retention window purges them for good.

## Profile
Update your display name or sign out from `Profile` in the sidebar.

## Everything is realtime
No refresh button needed — anything a teammate does (upload, rename, delete, comment, tag) appears on your screen within a second or two automatically.
